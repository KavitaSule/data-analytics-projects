# Data Analytics Projects

This repository contains hands-on data analytics projects completed as part of the UT Austin Data Analytics & Visualization program.

## Skills Used
- SQL (joins, CTEs, window functions)
- Python (Pandas, NumPy)
- Power BI (dashboards, KPIs, DAX)
- Data cleaning and analysis

## Projects
- FoodHub Analysis: customer behavior and revenue insights
- Business dashboards using Power BI
- Data exploration and reporting
